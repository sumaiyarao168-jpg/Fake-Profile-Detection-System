# Fake-Profile-Detection-System
Fake Profile Detection System

📌 Project Overview

The Fake Profile Detection System is a Python-based cybersecurity project that analyzes social media profiles using rule-based heuristics. It assigns a fake profile score based on account characteristics and classifies each profile as Legitimate, Suspicious, or Highly Suspicious.

This project was developed as part of my Cyber Security Internship at SQROCK IT SOLUTION.

🚀 Features

Detects potentially fake social media accounts.
Rule-based fake profile scoring system.
Evaluates multiple profile attributes.
Classifies profiles into three security levels.
Uses anonymized sample profile data.
Beginner-friendly Python implementation.

## 🛠️ Technologies Used

- Python 3.x
- Built-in Python Functions

---## 🔍 Detection Parameters

The system evaluates profiles using the following criteria:

- Account Age
- Followers-to-Following Ratio
- Profile Picture Availability
- Number of Posts
- Default Biography

Each factor contributes to an overall **Fake Profile Score (0–100).**

---

## 📊 Risk Classification

| Fake Score | Status |
|------------|-----------------------------|
| 70–100 | Highly Suspicious / Likely Bot |
| 40–69 | Suspicious / Potential Fake |
| 0–39 | Legitimate / Safe |
